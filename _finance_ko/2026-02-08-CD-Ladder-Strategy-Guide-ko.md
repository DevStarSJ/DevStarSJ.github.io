금리가 높을 때 정기예금(CD)은 매우 매력적입니다. 하지만 함정이 있습니다: 돈이 묶인다는 것. 높은 금리를 받으면서 **동시에** 정기적으로 자금에 접근할 수 있다면 어떨까요? 바로 **CD 래더 전략**입니다.

## CD 래더란?

CD 래더는 여러 개의 CD에 만기일을 분산하여 돈을 나눠 저축하는 전략입니다. 모든 돈을 하나의 장기 CD에 묶는 대신, 정기적인 간격으로 만기가 되는 CD의 "사다리"를 만듭니다.

### 기본 개념

$10,000을 저축한다고 가정해봅시다:

- **CD 1:** $2,000 → 1년 만기 → 1년 후 만기
- **CD 2:** $2,000 → 2년 만기 → 2년 후 만기
- **CD 3:** $2,000 → 3년 만기 → 3년 후 만기
- **CD 4:** $2,000 → 4년 만기 → 4년 후 만기
- **CD 5:** $2,000 → 5년 만기 → 5년 후 만기

매년 하나의 CD가 만기되어 자금에 접근할 수 있습니다. 돈을 사용하거나 현재 금리로 새 5년 CD에 재투자할 수 있습니다.

## 왜 CD 래더를 사용하나요?

### 1. 저축 계좌보다 높은 수익

CD는 일반적으로 일반 저축 계좌보다 높은 금리를 제공하며, 특히 장기일수록 그렇습니다.

**2026년 평균 금리:**
- 고금리 저축: 4.5% APY
- 1년 CD: 4.8% APY
- 5년 CD: 4.3% APY

### 2. 정기적인 유동성

하나의 큰 CD가 5년 후 만기되기를 기다리는 대신, 매년 (또는 구조에 따라 매월) 일부 자금에 접근할 수 있습니다.

### 3. 금리 변동 보호

금리가 오르면 만기된 CD를 더 높은 금리로 재투자할 수 있습니다. 금리가 내리면 장기 CD는 이전의 높은 금리로 고정되어 있습니다.

### 4. 예금자 보호

CD는 은행당, 예금자당 $250,000까지 보호됩니다. 원금이 완전히 안전합니다.

## 첫 CD 래더 만들기

### 1단계: 총 투자금 결정

CD 래더에 넣을 금액을 결정하세요. 이것은 비상금으로 필요하지 않은 돈이어야 합니다 (먼저 일반 저축 계좌에 3-6개월 생활비를 유지하세요).

### 2단계: 래더 길이 선택

일반적인 옵션:
- **5년 래더:** 수익 극대화에 최적
- **3년 래더:** 수익과 유연성의 균형
- **1년 래더:** 최대 유동성, 적당한 수익 증가

### 3단계: 균등하게 나누기

총 투자금을 래더의 단계 수로 균등하게 나눕니다.

**예시: 5년 래더로 $15,000**
- 5개 CD × 각 $3,000
- 기간: 1, 2, 3, 4, 5년

### 4단계: 최고 금리 찾기

다음에서 금리를 비교하세요:
- 온라인 은행 (보통 가장 높은 금리)
- 신용협동조합
- 전통 은행
- 투자 플랫폼을 통한 중개 CD

### 5단계: 만기 시 재투자

각 CD가 만기되면:
1. 현재 금리 평가
2. 자금이 필요한지 고려
3. 필요 없다면 래더에서 가장 긴 기간으로 재투자

## CD 래더 변형

### 미니 래더 (월별 접근)

더 많은 유동성을 위해 1년 CD로 12개월 래더 만들기:

- 1월: 12개월 CD 개설
- 2월: 12개월 CD 개설
- ... 매월 계속

12개월 후, 매달 하나의 CD가 만기됩니다!

### 바벨 전략

중간을 건너뛰고 단기와 장기 CD를 결합:

- 50%는 1년 CD
- 50%는 5년 CD

정기적인 유동성을 확보하면서 장기 금리도 잡을 수 있습니다.

### 불릿 전략

여러 CD가 동시에 만기—특정 날짜에 돈이 필요할 때 유용합니다 (주택 구매, 학비 납부 등).

## 실제 예시: 2026년에 $25,000 CD 래더 만들기

현재 금리가 다음과 같다고 가정:
- 1년 CD: 4.8%
- 2년 CD: 4.5%
- 3년 CD: 4.3%
- 4년 CD: 4.2%
- 5년 CD: 4.0%

- **1년:** $5,000 @ 4.8% → 첫해 이자 $240
- **2년:** $5,000 @ 4.5% → 첫해 이자 $225
- **3년:** $5,000 @ 4.3% → 첫해 이자 $215
- **4년:** $5,000 @ 4.2% → 첫해 이자 $210
- **5년:** $5,000 @ 4.0% → 첫해 이자 $200
- **총계:** $25,000, 평균 4.36%, 첫해 이자 $1,090

첫 해 후:
- 1년 CD 만기 ($5,240)
- 새 5년 CD에 재투자
- 이제 다시 1, 2, 3, 4, 5년 만기 CD 보유

## CD 래더 vs 다른 옵션

### CD 래더 vs 고금리 저축

- **수익:** CD 래더 더 높음
- **유동성:** 저축계좌가 즉시 접근 가능
- **금리 고정:** CD 래더만 가능
- **노력:** CD 래더가 더 필요

**결론:** 비상금은 고금리 저축, 중기 저축은 CD 래더.

### CD 래더 vs 국채

둘 다 안전하지만:
- 국채가 현재 약간 낮은 금리
- 국채는 주 세금 면제
- CD가 관리하기 더 간단

### CD 래더 vs 채권 펀드

- CD는 원금 위험 없음
- 채권 펀드는 금리 상승 시 가치 하락 가능
- CD는 예금자 보호; 채권 펀드는 아님

## 피해야 할 흔한 실수

### 실수 1: 조기 인출 페널티 무시

개설 전에 페널티를 확인하세요. 일반적인 페널티:
- 1년 CD: 3개월 이자
- 2-5년 CD: 6개월 이자

### 실수 2: 비교하지 않음

기관마다 금리가 크게 다릅니다. 항상 5-10개 옵션을 비교하세요.

### 실수 3: 세금 잊음

CD 이자는 일반 소득으로 과세됩니다. 계획에 이를 고려하세요.

### 실수 4: 모든 저축 사용

CD 래더를 만들기 전에 유동성 있는 저축에 비상금을 유지하세요.

## 2026년 CD 래더를 위한 최고의 은행

다음을 찾으세요:
- 경쟁력 있는 금리
- 낮거나 없는 최소 예치금
- 합리적인 조기 인출 페널티
- 쉬운 온라인 관리

**상위 온라인 옵션:**
- Ally Bank
- Marcus by Goldman Sachs
- Discover Bank
- Synchrony Bank
- Capital One

## CD 래더를 사용하지 않을 때

다음의 경우 CD 래더가 이상적이지 않을 수 있습니다:
- 모든 돈이 예기치 않게 필요할 수 있음
- 인플레이션이 CD 금리보다 높음
- 주식 시장 변동성을 감수할 수 있음 (장기 수익이 더 높음)
- 매우 높은 세금 구간 (대신 지방채 고려)

## 오늘 시작하기 위한 실행 단계

1. **투자할 금액 계산** (비상금 확보 후)
2. **래더 길이 결정** (3년 또는 5년 권장)
3. **온라인 은행에서 현재 금리 조사**
4. **오늘 첫 번째 단계 개설**
5. **만기일에 캘린더 알림 설정**
6. **래더를 추적할 간단한 스프레드시트 만들기**

## 결론

CD 래더는 합리적인 자금 접근성을 유지하면서 저축에서 더 높은 수익을 얻는 가장 간단하고 안전한 방법 중 하나입니다. 고금리 환경에서는 특히 강력한 도구입니다.

부자가 되게 해주지는 않지만, 일반 저축 계좌보다 저축이 더 열심히 일하게 해줍니다—거의 제로 위험으로.

오늘 래더를 만들기 시작하세요, 한 단계씩.
