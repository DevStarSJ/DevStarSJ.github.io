# 제로 베이스 예산법: 모든 돈에 역할을 부여하고 재정을 장악하는 방법

대부분의 사람들은 통장 잔액을 확인하고 월말에 남아있길 바라는 것으로 "예산 관리"를 합니다. 그건 예산이 아니라 기도입니다.

**제로 베이스 예산법(Zero-Based Budgeting, ZBB)**은 정반대입니다. 돈이 어디로 갔는지 궁금해하는 대신, 월초에 *미리* 어디로 갈지 정합니다. 모든 돈에 역할을 부여합니다. 수입 마이너스 지출이 0이 됩니다.

![계산기와 재정 계획](https://images.unsplash.com/photo-1554224155-8d4a0b14db72?w=800)
*Photo by [Scott Graham](https://unsplash.com/@homajob) on Unsplash*

딱딱해 보이지만, 실제로는 해방감을 줍니다.

## 제로 베이스 예산법이란?

**수입에서 모든 계획된 지출을 빼면 0이 되는 것**입니다. 빈털터리여서가 아니라, 모든 돈에 목적이 있기 때문입니다.

**전통적 예산법:** "벌이보다 적게 쓰도록 노력하자."
**제로 베이스 예산법:** "이번 달 수입 400만원. 각 항목에 이렇게 배분한다."

공식:

```
수입 - (필수 + 원하는 것 + 저축 + 부채상환) = 0원
```

20만원이 남았다면? 그건 "여유분"이 아닙니다 — 배정하세요. 부채 상환, 저축, 또는 다음 달 예산으로. ZBB에서 미배정 돈은 낭비된 가능성입니다.

## 제로 베이스 예산법이 효과적인 이유

### 자각을 만들어줍니다
대부분 돈의 30~40%가 어디로 가는지 모릅니다. ZBB는 모든 지출과 마주하게 합니다. 매일 5,000원짜리 커피? 보입니다. 잊고 있던 월 4만원 구독료? 그것도 보입니다.

### 지출을 가치에 맞춥니다
모든 돈을 펼쳐놓으면 자연스럽게 중요한 것에 우선순위를 둡니다. 외식에 월 80만원을 쓰면서 실제로 행복을 주는 취미에 5만원만 쓰고 있다는 걸 깨달을 수 있습니다.

### 생활 수준 인플레이션을 방지합니다
연봉이 오르면? 전통적 예산법에서는 추가 소득이 어디론가 사라집니다. ZBB에서는 추가분을 **의도적으로** 배정합니다.

### 비정기 소득에도 적합합니다
프리랜서, 영업직, 긱워커들이 ZBB를 좋아하는 이유입니다. 이번 달 300만원? 300만원으로 예산. 다음 달 700만원? 700만원으로 예산.

## 제로 베이스 예산 설정법

### 1단계: 수입 정리

다음 달의 모든 수입원을 적으세요:
- 급여 (세후)
- 부업 수입
- 프리랜서 수입
- 투자 수익
- 기타 수입

**수입이 불규칙하다면:** 보수적으로 가장 적을 것 같은 금액을 기준으로 잡고, 추가 수입이 들어올 때 배정하세요.

### 2단계: 모든 지출 나열

고정 지출부터, 그다음 변동 지출:

**고정 (매월 동일):**
- 월세/대출 상환
- 보험료
- 통신비
- 구독 서비스
- 최소 부채 상환

**변동 (매월 변동):**
- 식비/장보기
- 교통비
- 공과금
- 여가/문화
- 외식
- 의류
- 미용/관리

**비정기 (매월 발생하지 않음):**
- 차량 유지비
- 의료비
- 경조사비
- 연간 구독
- 여행 적립

### 3단계: 모든 돈 배정

총 수입에서 지출을 빼서 0이 될 때까지 배정합니다.

![노트에 메모하는 사람](https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?w=800)
*Photo by [Cathryn Lavery](https://unsplash.com/@cathrynlavery) on Unsplash*

**예시: 월 수입 400만원**

| 항목 | 금액 |
|------|------|
| **필수** | |
| 월세 | 70만원 |
| 식비/장보기 | 40만원 |
| 공과금 | 15만원 |
| 교통비 | 10만원 |
| 보험료 | 15만원 |
| 통신비 | 6만원 |
| **원하는 것** | |
| 외식 | 20만원 |
| 여가/문화 | 10만원 |
| 의류 | 7만원 |
| 구독 서비스 | 3만원 |
| **재정 목표** | |
| 비상금 적립 | 40만원 |
| 연금저축 | 30만원 |
| 투자 (ETF 적립) | 20만원 |
| 여행 적립 | 10만원 |
| **여유분/경조사** | 4만원 |
| **합계** | **400만원** |

수입 (400만원) - 배정 합계 (400만원) = **0원** ✅

### 4단계: 월중 추적

추적하지 않는 예산은 소원 목록일 뿐입니다. 최소 주 1회 실제 지출을 계획과 비교하세요.

**추적 도구:**
- **뱅크샐러드** — 자동 분류, 무료
- **토스** — 소비 분석 기능
- **네이버 가계부** — 수동 입력
- **엑셀/구글 시트** — 무료, 맞춤 가능
- **수기 가계부** — 아날로그의 힘

### 5단계: 필요에 따라 조정

인생은 계획대로 되지 않습니다. 예산은 계획이지 감옥이 아닙니다. 한 항목에서 초과하면 다른 항목을 줄이세요. 목표는 여전히: **수입 - 지출 = 0**.

식비에서 5만원 초과? 외식이나 의류에서 5만원 빼세요. 합계는 맞아야 합니다.

## 50/30/20 프레임워크

배분 방법을 모르겠다면 **50/30/20 규칙**으로 시작하세요:

- **50% 필수:** 주거, 식비, 교통, 보험, 최소 부채 상환
- **30% 원하는 것:** 여가, 외식, 취미, 비필수 쇼핑
- **20% 저축과 투자:** 비상금, 연금, 투자, 추가 부채 상환

여기서 커스터마이즈하세요. 공격적인 재정 목표가 있다면 40/10/50이나 50/10/40으로 조정.

## 흔한 제로 베이스 예산 실수

### ❌ 비정기 지출 잊기
6개월마다 오는 보험료? 월할로 나눠 적립하세요. 명절 경조사비, 차량 검사비, 연간 구독 — 모두 월별로 분산.

### ❌ "기타" 항목 빼먹기
어떤 예산도 100% 현실과 맞지 않습니다. 소액 예비비(5~10만원)를 포함하세요.

### ❌ 너무 빡빡하게 짜기
여가비 0원인 예산은 일주일도 못 갑니다. 즐길 거리도 예산에 넣되, 의도적으로 정하세요.

### ❌ (커플의 경우) 혼자서 짜기
재정을 공유한다면 둘 다 예산에 동의해야 합니다. 월 1회 "예산 회의" (커피 마시며 15분)가 돈 싸움을 예방합니다.

### ❌ 한 달 실패로 포기하기
어떤 달은 초과합니다. 그건 실패가 아니라 데이터입니다. 조정하고 다시 하세요. ZBB에 익숙해지는 데 2~3개월 걸립니다.

## 한국에서 특히 신경 쓸 항목

### 경조사비
한국 특유의 큰 변동 지출입니다. 월 평균 10~20만원을 경조사 적립으로 따로 빼두세요.

### 연말정산/세금
연말정산 환급이나 추가 납부를 예산에 반영하세요. 연금저축, IRP 납입은 절세 효과도 있어 예산에 우선 배치하면 일석이조입니다.

### 명절 비용
설/추석에 큰 지출이 발생합니다. 월 5~10만원씩 명절 적립금을 만들어두세요.

### 통신비 최적화
알뜰폰(MVNO)으로 바꾸면 월 3~5만원 절약 가능. ZBB를 시작하면 이런 고정비 절약 포인트가 보입니다.

## ZBB가 편해지는 시점

처음 2개월은 일처럼 느껴집니다. 3개월째부터 루틴이 됩니다. 6개월 후에는 이전으로 돌아갈 수 없습니다.

- **1개월:** 압도적. 항목을 빠뜨릴 겁니다. 괜찮습니다.
- **2개월:** 나아짐. 패턴이 보입니다. 조정이 자연스러워집니다.
- **3개월:** 감이 옵니다. 다음 달 예산 짜는데 15분.
- **6개월+:** 자동화됨. 꺼지지 않는 재정 감각이 생깁니다.

## 결론

제로 베이스 예산법은 제한이 아니라 **의도**입니다. 돈이 나를 지배하는 것과 내가 돈을 지배하는 것의 차이입니다.

완벽할 필요 없습니다. **의도적**이면 됩니다. 모든 돈에 역할을 주고, 진행을 추적하고, 가면서 조정하세요.

**이번 달부터 시작하세요. 스프레드시트를 열고. 수입을 적고. 모든 돈을 배정하고. 0을 맞추세요.**
