# Perplexity AI: 2026년 검색의 미래

답을 찾으려고 파란 링크 10개를 스크롤하는 것에 지치셨나요? Perplexity AI가 인터넷 검색 방식을 바꾸고 있습니다. 단순한 챗봇이 아닙니다 — **출처를 인용하는 AI 기반 답변 엔진**입니다.

## Perplexity가 특별한 이유

### 1. 실시간 웹 검색 + AI

ChatGPT(지식 컷오프가 있음)와 달리, Perplexity는 실시간으로 웹을 검색하고 정보를 종합합니다. 오늘의 뉴스, 주가, 최신 연구에 대해 물어보세요 — 현재 정보를 얻을 수 있습니다.

### 2. 모든 것에 출처 표시

모든 답변에 번호가 매겨진 인용이 포함됩니다. 클릭해서 검증하세요. 더 이상 "이게 어디서 나온 거지?" 고민할 필요 없습니다. 이 기능은 다음에 매우 유용합니다:

- 학술 연구
- 팩트체킹
- 실사(Due Diligence)
- 저널리즘

### 3. 포커스 모드

특화된 모드로 검색을 타겟팅하세요:

| 모드 | 최적 용도 |
|------|----------|
| **All** | 일반 질문 |
| **Academic** | 연구 논문 & 학술 자료 |
| **Writing** | 콘텐츠 작성 도움 |
| **Wolfram** | 수학 & 계산 |
| **YouTube** | 영상 기반 답변 |
| **Reddit** | 커뮤니티 의견 & 토론 |

## 실제 사용 사례

### 개발자용

```
"React 18에서 useEffect와 useLayoutEffect의 차이점은?"
```

Perplexity가 공식 React 문서, Stack Overflow 토론, 최근 블로그 포스트에서 정보를 가져옵니다 — 모두 출처와 함께.

### 연구자용

```
"2025-2026년에 발표된 간헐적 단식과 수명에 관한 최신 연구"
```

Academic 모드를 사용해 직접 링크가 포함된 피어 리뷰 논문을 얻으세요.

### 비즈니스용

```
"한국에서 소규모 비즈니스 결제 처리를 위한 Stripe vs Square 비교"
```

최신 가격, 기능, 지역별 이용 가능 여부를 확인하세요.

## 파워 유저를 위한 프로 팁

### 1. 컬렉션 사용하기

관련 검색을 컬렉션에 저장하세요. 프로젝트를 진행 중이라면? 컬렉션을 만들면 Perplexity가 검색 간에 컨텍스트를 기억합니다.

### 2. 문서 업로드

Pro 사용자는 PDF, 코드 파일, 문서를 업로드하고 웹 컨텍스트와 함께 질문할 수 있습니다.

### 3. 다른 모델 시도하기

Pro에서 전환 가능:
- **Sonar** (Perplexity 자체 모델, 가장 빠름)
- **GPT-4o** (복잡한 추론에 최적)
- **Claude** (뉘앙스 있는 글쓰기에 최적)

### 4. 연산자 사용하기

구글처럼 사용할 수 있습니다:
- `site:github.com` — 특정 사이트 검색
- `"정확한 문구"` — 정확히 일치
- `-제외` — 용어 제외

## 개발자를 위한 API

Perplexity는 놀랍도록 저렴한 API를 제공합니다:

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-perplexity-key",
    base_url="https://api.perplexity.ai"
)

response = client.chat.completions.create(
    model="sonar-pro",
    messages=[
        {"role": "user", "content": "최신 Next.js 15 기능"}
    ]
)
```

가격: Sonar 모델 기준 100만 토큰당 ~$5 (GPT-4보다 훨씬 저렴).

**예, 만약:**
- 정기적으로 리서치를 한다면
- 작업에 출처가 필요하다면
- 현재 정보를 원한다면
- 구글 검색 결과의 SEO 최적화된 글에 지쳤다면

**아마 아닐 수도, 만약:**
- 빠른 계산만 필요하다면 (Wolfram 사용)
- 창의적인 AI 대화를 원한다면
- 예산이 빠듯하고 무료 티어 제한이 신경 쓰인다면
