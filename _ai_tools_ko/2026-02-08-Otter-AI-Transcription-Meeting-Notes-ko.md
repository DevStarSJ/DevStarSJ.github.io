## Otter.ai란?

Otter.ai는 **AI 기반 음성 인식 및 회의 어시스턴트**입니다. 자동으로 미팅을 녹음, 기록, 요약합니다. Zoom, Google Meet, Teams 회의에 참가자로 참여하여 회의 종료 몇 분 내에 검색 가능한 노트를 전달합니다.

### 팀들이 Otter로 바꾸는 이유

- **실시간 음성 인식**: 회의 중 라이브 자막
- **자동 참여**: Otter가 캘린더 일정에 자동 참여
- **화자 식별**: 누가 뭘 말했는지 파악
- **AI 요약**: 핵심 포인트와 액션 아이템 추출
- **전체 검색**: 키워드로 모든 회의 찾기

## 주요 기능

### 1. OtterPilot

AI 회의 어시스턴트 기능:

- 예약된 회의에 자동 참여
- 오디오 및 비디오 녹음
- 실시간 음성 인식
- 다른 화자 식별
- 회의 후 요약 생성

### 2. 실시간 음성 인식

사람들이 말하는 대로 텍스트 표시:

```
[10:03 AM] 김철수: Q2 마케팅 예산 논의하죠. 
소셜 미디어 지출을 20% 늘려야 할 것 같아요.

[10:03 AM] 이영희: 야심 차네요. 그 증가에 
예상 ROI가 어떻게 되나요?

[10:04 AM] 김철수: Q1 데이터 기준으로, 
유료 소셜 캠페인에서 3배 수익을 보고 있어요.
```

### 3. AI 기반 요약

![회의에서 노트 작성](https://images.unsplash.com/photo-1517245386807-bb43f82c33c4?w=800)
*Photo by [You X Ventures](https://unsplash.com/@youxventures) on Unsplash*

모든 회의 후, Otter가 생성:

- **요약**: 3-5문장 개요
- **핵심 포인트**: 불릿 포인트 하이라이트
- **액션 아이템**: 누가 뭘 해야 하는지
- **키워드**: 검색 가능한 태그

### 4. 협업 기능

- **기록 공유**: 팀원에게 전송
- **하이라이트 & 코멘트**: 중요 섹션 표시
- **함께 편집**: 협업 편집
- **내보내기**: PDF, Word, SRT, TXT 형식

### 교육

- **강의 녹음**: 학생들이 기록에 접근
- **오피스 아워**: Q&A 세션 녹음

### 1. 화자 식별 최적화

- 음성 등록 기능 사용

### 2. 프로처럼 검색

Otter 검색 연산자 사용:

```
"제품 출시"         → 정확한 구문
speaker:철수        → 특정 사람
date:2024-03-01    → 특정 날짜
액션 아이템          → 할 일 찾기
```
