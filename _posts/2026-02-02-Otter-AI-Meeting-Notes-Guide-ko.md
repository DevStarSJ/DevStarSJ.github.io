# Otter.ai: 더 이상 회의록 작성 안 해도 됩니다

회의 중에 듣기도 하고, 메모도 하고, 질문도 생각해야 합니다. 잘 안 됩니다. **Otter.ai**가 대신 듣고, 모든 것을 받아쓰고, 중요한 부분을 요약해줍니다.

![회의실](https://images.unsplash.com/photo-1552664730-d307ca884978?w=800)

## Otter.ai가 하는 일

- **실시간 트랜스크립션** - 말하는 대로 텍스트 표시
- **화자 식별** - 누가 말했는지 구분
- **자동 요약** - 핵심 포인트 추출
- **액션 아이템** - 할 일 자동 감지
- **회의 연동** - Zoom/Teams/Meet 자동 참가

## 시작하기

### 1. 계정 생성

[otter.ai](https://otter.ai)에서 가입. 무료로 월 300분 제공.

### 2. 캘린더 연결

Google Calendar 또는 Outlook 연결:
- Otter가 예정된 회의 확인
- 회의 시작 시 자동 참가
- 수동 설정 불필요

### 3. 앱 설치

- **데스크탑 앱** - 오프라인 회의용 (노트북 마이크 사용)
- **모바일 앱** - 이동 중 녹음
- **Chrome 확장** - 트랜스크립트 빠른 접근

## 실제 작동 방식

### 화상 회의

Zoom 통화 전:

1. Otter가 캘린더에서 회의 확인
2. "OtterPilot" 봇이 자동 참가
3. 봇이 전체 회의 트랜스크립션
4. 몇 분 내 요약 이메일 수신

아무것도 안 해도 됩니다. 그냥 작동합니다.

### 오프라인 회의

Otter 앱을 열고 녹음 시작:

1. 모두가 들리는 곳에 폰/노트북 배치
2. Otter가 실시간 트랜스크립션
3. 탭으로 하이라이트 추가 가능
4. 필요시 나중에 트랜스크립트 편집

## 핵심 기능

### OtterPilot (자동 참가)

킬러 기능. 한 번만 설정:

1. 설정 → OtterPilot 이동
2. "Auto-join meetings" 활성화
3. 참가할 캘린더 이벤트 선택
4. 알림 설정

OtterPilot이 참가하고, 자기소개하고, 트랜스크립션하고, 나갑니다. 참석자들은 참가자로 봇을 봅니다.

### 화자 식별

Otter가 시간이 지나며 목소리 학습:

- 첫 회의: "Speaker 1", "Speaker 2"
- 누가 누군지 라벨링
- 이후 회의: 이름 자동 할당

정기 회의 참석자를 트레이닝하면 빠르게 정확해집니다.

### AI 채팅

회의에 대해 질문:

```
"사라가 예산에 대해 뭐라고 했어?"
"언급된 모든 마감일 목록 보여줘"
"디자인에 대한 존의 피드백 요약해줘"
```

AI가 모든 트랜스크립트를 검색하고 타임스탬프와 함께 답변합니다.

### 액션 아이템

누군가 약속할 때 Otter가 감지:

> "금요일까지 제안서 보내드릴게요"

이게 액션 아이템이 됩니다:
- **담당자:** 화자
- **태스크:** 제안서 발송
- **기한:** 금요일

액션 아이템은 쉬운 후속 조치를 위해 별도 섹션에 수집됩니다.

### 협업 편집

팀과 트랜스크립트 공유:

- 중요한 섹션 하이라이트
- 코멘트 추가
- 트랜스크립션 오류 수정
- 특정 순간의 클립 생성

팀 모두가 접근하고 기여할 수 있습니다.

## 연동 가이드

### Zoom

1. Otter 설정에서 Zoom 계정 연결
2. Zoom용 OtterPilot 활성화
3. Otter 봇이 모든 Zoom 회의에 자동 참가

예약된 회의와 즉석 회의 모두 작동.

### Microsoft Teams

1. Teams 앱 스토어에서 Otter 설치
2. 연결 인증
3. OtterPilot이 Teams 회의 참가

참고: 일부 조직에서는 Teams 관리자 승인 필요.

### Google Meet

1. Chrome 확장 설치
2. Google 계정 연결
3. Meet 통화 중 Otter 아이콘 클릭

Meet 연동은 확장 프로그램 필요; 봇 옵션 없음.

### Slack

트랜스크립트를 채널에 직접 공유:

1. Slack 워크스페이스 연결
2. 회의 후 "Share to Slack" 클릭
3. 채널 선택
4. 요약과 액션 아이템 자동 게시

## 영업팀용 Otter

영업팀을 위한 특별 기능:

- **딜 인사이트** - 경쟁사, 가격, 이의 언급 추적
- **코칭** - 매니저와 통화 리뷰
- **CRM 동기화** - Salesforce/HubSpot에 노트 푸시
- **말하기 비율** - 누가 가장 많이 말했는지 (힌트: 고객이 말하게 하세요)

영업 리더들은 참석하지 않고도 모든 통화를 리뷰할 수 있어 좋아합니다.

## 프라이버시 고려사항

알아야 할 중요한 것들:

- **녹음 동의** - 많은 관할권에서 필요
- **봇 공지** - OtterPilot이 녹음 중임을 말함
- **데이터 보관** - 트랜스크립트 보관 기간 제어
- **암호화** - 전송 중 및 저장 시 데이터 암호화
- **컴플라이언스** - SOC 2 Type II, GDPR, HIPAA (Business 플랜)

항상 참가자에게 회의가 녹음되고 있음을 알려주세요.

## 일반적인 문제 & 해결책

### "OtterPilot이 회의에 참가 안 함"

- 캘린더 연결 확인
- 회의에 비디오 링크 있는지 확인
- 해당 캘린더에 OtterPilot 활성화 확인

### "화자 식별이 틀림"

- 수동으로 화자 수정 및 라벨링
- 시간 지나며 음성 프로필 구축
- 좋은 오디오 품질 보장

### "트랜스크립션에 오류 있음"

- 재처리를 위해 오디오 파일 업로드
- 트랜스크립트 수동 편집
- 산업별 어휘 트레이닝
